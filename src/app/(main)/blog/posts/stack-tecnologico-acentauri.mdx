En αCentauri, creemos que elegir el stack tecnológico correcto es fundamental para el éxito de cualquier startup. Aquí te compartimos las tecnologías que utilizamos y por qué las elegimos.

## Frontend: Next.js y React

Next.js es nuestra elección principal para el frontend por varias razones:

- **Server-Side Rendering**: Mejor SEO y rendimiento inicial
- **File-based routing**: Organización clara del código
- **Optimización automática**: Imágenes, fuentes y código optimizados sin configuración

## Backend: Node.js y TypeScript

Para el backend, confiamos en Node.js con TypeScript:

- **Mismo lenguaje**: JavaScript/TypeScript en todo el stack
- **Ecosistema robusto**: Miles de paquetes disponibles en npm
- **Alto rendimiento**: Ideal para aplicaciones en tiempo real

## Base de Datos: PostgreSQL y MongoDB

Dependiendo del proyecto, utilizamos:

- **PostgreSQL**: Para datos estructurados y transaccionales
- **MongoDB**: Para datos flexibles y de rápido crecimiento

## Infrastructure: Vercel y AWS

Desplegamos nuestras aplicaciones en:

- **Vercel**: Hosting optimizado para Next.js con CI/CD automático
- **AWS**: Para servicios más complejos que requieren mayor control

## Design System: Once UI

Utilizamos Once UI, un sistema de diseño moderno que nos permite:

- Desarrollo rápido con componentes pre-construidos
- Consistencia visual en todos los proyectos
- Accesibilidad y responsive design por defecto

## ¿Por qué este Stack?

Este stack nos permite:

1. **Velocidad de desarrollo**: Iteramos rápidamente
2. **Escalabilidad**: Crece con tu negocio
3. **Mantenibilidad**: Código limpio y organizado
4. **Developer Experience**: Herramientas modernas que mejoran la productividad

En αCentauri, no solo elegimos tecnologías populares, sino aquellas que resuelven problemas reales y permiten a las startups paraguayas competir a nivel global.
